设计函数 locatesubstr
题目描述
查找子串在字符串中首次出现的位置，返回指针。

函数接口
c
char locatesubstr(char str1, char str2);
裁判测试程序样例
c
#include stdio.h
char locatesubstr(char str1,char str2);
int main()
{
    char str1[505],str2[505];
    char p;
    gets(str1);
    gets(str2);
    p=locatesubstr(str1,str2);
    if(p==NULL) printf(NULL!n);
    else puts(p);
    return 0;
}
输入样例
text
didjfsd dabcxxxxxx
abc
输出样例
text
abcxxxxxx


char *locatesubstr(char *str1, char *str2){
  if(*str2=='\0')return str1;
  while(*str1){
    //这里要写str1,因为要走完，防止有多处相似结构
    char *s1=str1;
    char *s2=str2;
    while(*s1&&*s2&&*s1==*s2){

	//这里要注意s1,s2都不能走完了
      s1++;
      s2++;
    }
    if(*s2=='\0')return str1;
    //出口

    str1++;
    //实际上是在改变str1指针指向的位置，不是指向原来第一个
	//但内容没变
  }
  return NULL;
}

关键要点
指针操作：用两个指针同步移动比较。

子串匹配：内层循环比较直到不等或子串结束。

返回：若子串结束，则找到，返回起始地址。
