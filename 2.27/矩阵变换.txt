矩阵变换
题目描述
按行平均值升序重排行，平均值相同则保持原顺序。

输入格式
第一行 n，接着 n 行 n 列整数。

输出格式
重排后的矩阵。

样例输入
text
3
5 5 5
3 3 3
1 1 1
样例输出
text
1 1 1
3 3 3
5 5 5
参考代码
c
#includestdio.h
int main(){
    int n;
    scanf(%d,&n);
    int a[n][n];
    double avg[n];
    int order[n];
    for(int i=0;in;i++){
        int sum=0;
        for(int j=0;jn;j++){
            scanf(%d,&a[i][j]);
            sum += a[i][j];
        }
        avg[i] = (double)sum  n;
        order[i] = i;
    }
     冒泡排序 order 数组
    for(int i=0;in-1;i++){
        for(int j=0;jn-1-i;j++){
            if(avg[order[j]]  avg[order[j+1]]){
                int temp = order[j];
                order[j] = order[j+1];
                order[j+1] = temp;
            }
        }
    }
    for(int i=0;in;i++){
        int row = order[i];
        for(int j=0;jn;j++){
            if(j==0) printf(%d,a[row][j]);
            else printf( %d,a[row][j]);
        }
        printf(n);
    }
    return 0;
}


或者直接一个一个交换数组的元素
关键要点
稳定排序：用 order 数组记录原始行号，按平均值排序，不交换原数组。

平均值相同：冒泡排序不交换相等元素，保持原序。
