void bubbleSort(int a[], int n){
    for(int i=0; in-1; i++){
        for(int j=0; jn-i-1; j++){
            if(a[j]  a[j+1]){
                int temp = a[j];
                a[j] = a[j+1];
                a[j+1] = temp;
            }
        }
    }
}

关键要点
冒泡排序：相邻比较，每一趟把最大数沉到末尾。

输出格式：第一个数前无空格，其余有空格，最后换行。
