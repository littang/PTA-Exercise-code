字符串排序（指针数组）
题目描述
动态创建指针数组和字符数组，读入 n 个字符串，按 ASCII 码升序排序后输出。

函数接口
c
char  create1( int n );
char  create2( int n );
void sort( char strArray , int size );
裁判测试程序样例
略。

输入样例
text
5
bbb
zzzzzz
aabbbccc
aaaaaa
abbbbb
输出样例
text
aaaaaa
aabbbccc
abbbbb
bbb
zzzzzz
参考代码
c
char  create1(int n){
    return (char)malloc(sizeof(char)n);
}
char  create2(int n){
    return (char)malloc(sizeof(char)n);
}
void sort(char strArray, int size){
    for(int i=0; isize-1; i++){
        for(int j=0; jsize-1-i; j++){
            if(strcmp(strArray[j], strArray[j+1])  0){
                char temp = strArray[j];
                strArray[j] = strArray[j+1];
                strArray[j+1] = temp;
            }
        }
    }
}
关键要点
动态分配：先分配指针数组，再为每个指针分配字符数组。

排序：用 strcmp 比较字符串，交换指针。

内存释放：在 main 中负责释放。
