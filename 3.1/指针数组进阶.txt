指针数组进阶
题目描述
连续存储所有字符串于一个一维字符数组中，用指针数组指向每个字符串的起始位置，然后排序输出。

输入格式
第一行 n，后面 n 行每行一个字符串（可能含空格）。

输出格式
排序后的 n 行字符串。

样例输入
text
4
Where there is hope ,there is a way.
Welcome Beijing.
Nice idea.
Have fun.
样例输出
text
Have fun.
Nice idea.
Welcome Beijing.
Where there is hope ,there is a way.

#include<stdio.h>
#include<stdlib.h>
#include<string.h>
int main(){
    int n;
    scanf("%d",&n);
    getchar();
    char**str=(char**)malloc(sizeof(char*)*n);
    for(int i=0;i<n;i++){
        char stra[1024];
        gets(stra);
        int len=strlen(stra);
        str[i]=(char*)malloc(sizeof(char)*(len+1));
        strcpy(str[i],stra);
    }
    for(int i=0;i<n;i++){
        for(int j=0;j<n-1-i;j++){
            if(strcmp(str[j],str[j+1])>0){
                char* a=str[j];
                str[j]=str[j+1];
                str[j+1]=a;
            }
        }
    }
    for(int i=0;i<n;i++){
        puts(str[i]);
    }
    for(int i=0;i<n;i++){
        free(str[i]);
    }
    free(str);
    return 0;
}
