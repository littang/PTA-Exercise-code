题目描述
用 -（减号）、|（竖线）和空格打印方格。给定格子的行数 l 和列数 c，打印对应的方格图形。

输入格式
第一行一个整数 t（0<t<10），表示测试用例组数。后面 t 行，每行两个整数 l 和 c（0<l,c<10），分别代表格子的行数和列数。

输出格式
每组测试用例输出一组方格，格式如样例所示。

样例输入
text
3
2 3
3 2
4 5
样例输出
text

|  |  |  |

+-----+
|  |  |
|  |  |
|-----|
|  |  |
|  |  |
|-----|
|  |  |
|  |  |
+-----+
+--------------+
|  |  |  |  |  |
|  |  |  |  |  |
|--------------|
|  |  |  |  |  |
|  |  |  |  |  |
|--------------|
