身份证号码最后一位
题目描述
根据前17位计算校验码，判断身份证最后一位是否正确。

输入格式
第一行整数 T（0<T<100），后面 T 行每行一个18位身份证号（可能含X）。

输出格式
每行输出 right 或 wrong。

样例输入
text
2
34052419800101001X
310105199412049278
样例输出
text
right
wrong
参考代码
c
#include<stdio.h>
int main(){
    char a[20];
    int xs[]={7,9,10,5,8,4,2,1,6,3,7,9,10,5,8,4,2};
    char m[]={'1','0','X','9','8','7','6','5','4','3','2'};
    int n;
    scanf("%d",&n);
    getchar();
    for(int i=0;i<n;i++){
        gets(a);
        int sum=0;
        for(int j=0;j<17;j++){
            sum+=(a[j]-'0')*xs[j];
        }
        sum%=11;
        char su = m[sum];    
        if (su == a[17]) printf("right\n");
        else printf("wrong\n");
    }
    return 0;
}
关键要点
字符转数字：a[j]-'0' 将字符数字转换为整数。

系数表：固定系数数组。

校验码表：余数0~10对应 '1','0','X','9',...,'2'。

输入处理：用 getchar() 吸收换行符，避免影响 gets。
