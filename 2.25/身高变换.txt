
7-4 身高变换
题目描述
完成英制和公制长度单位的转换。输入每行第一个整数表示转换方向：1 表示英制转公制，后跟英尺和英寸；0 表示公制转英制，后跟丝米数（万分之一米）。

输入格式
第一行为整数 n，表示有 n 个数据需要转换。接下来 n 行，每行有 2 或 3 个用空格分隔的整数，格式如题。

输出格式
共 n 行，每行对应转换结果。公制输出浮点数（保留两位小数），英制输出三个整数：英尺、英寸和剩余丝米数。

样例输入
text
5
1 4 3
1 6 5
0 17239
1 3 7
0 19567
样例输出
text
1.30
1.96
5 7 221
1.09
6 5 9



#include<stdio.h>
int main(){
  int n;
  scanf("%d",&n);
  for(int i=0;i<n;i++){
    int way;
    scanf("%d",&way);
    if(way){
      double inch,inc;
      scanf("%lf%lf",&inch,&inc);
      printf("%.2lf\n",inch*0.3048+inc*0.0254);
    }
    else {
      int wim;
      scanf("%d",&wim);
      printf("%d %d %d\n",wim/3048,wim%3048/254,wim%254);
    }
  }
  return 0;
}
