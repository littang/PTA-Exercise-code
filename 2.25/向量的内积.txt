7-5 实验7_4_向量的内积
题目描述
计算两个长度为 n 的一维向量的内积。

输入格式
第一行一个整数 n（1≤n≤10），第二行 2n 个整数（以空格分隔），即两个向量。

输出格式
一个整数，为内积结果。

样例输入
text
3
1 2 3 4 5 6
样例输出
text
32

#include<stdio.h>
int main(){
  int n;
  scanf("%d",&n);
  int a[n];
  for(int i=0;i<n;i++){
    scanf("%d",&a[i]);
  }
  int sum=0;
  for(int i=0;i<n;i++){
    int aa;
    scanf("%d",&aa);
    sum+=a[i]*aa;
  }
  printf("%d",sum);
  return 0;
}
